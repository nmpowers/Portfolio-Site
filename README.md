# Nathaniel Powers - Personal Portfolio

## Overview
This repository contains the source code for my personal portfolio website, a mono-page application aimed at prospective recruiters and collaborators. The site is designed to convey information about my projects, skills, and personality in a simple, visual, and understandable manner.

Because I do not anticipate needing to update the site more than once a week, it is built as a static site intended to be hosted freely on GitHub Pages. Future iterations may include more in-depth backend implementations using a database like Firebase if needed.

Visit the site here -> https://nmpowers.github.io/Portfolio-Site/

## Tech Stack
The portfolio was originally designed using Figma and developed using the following technologies:

- React
- TypeScript
- Vite
- Tailwind CSS
- Shadcn components

## Site Structure & Features
The application is built as a seamless mono-page experience containing the following distinct sections:

- **Navigation Bar**: A sticky header that provides smooth scrolling links to each section of the page. It includes a mobile-responsive dropdown menu and a quick-access button to download my resume.
- **Hero Section**: A dynamic, automated image slider showcasing different facets of my identity, including my roles as a leader, researcher, engineer, runner, hiker, and more.
- **About Me (Bio)**: An introduction detailing my background as a double major in Computer Science and Mechanical Engineering at WPI. It highlights my core interests and direction.
- **Timeline**: A chronological breakdown of my professional experience and academic leadership. This includes roles in internships in addition to extracurricular activities such as Student Government President, etc.
- **Projects Gallery**: An interactive grid of project cards displaying my recent technical and architectural work. Clicking on a project opens a detailed dialog containing a full description, an image gallery, the technologies used, and quick links to live demos, research papers, or GitHub repositories.
- **Contact**: A footer section providing quick links to connect with me via Email, GitHub, LinkedIn, and Instagram.
